## Update cloud resource annotations



Update cloud resource annotations. The request must contain either a contact email or id for each contact and tag name or id for each tag. If both contact email and id are supplied, only the id will be used. If tag name and id are supplied only the id will be used. If the tag name or contact email do not exist yet, they will be created.
